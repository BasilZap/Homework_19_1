# Домашняя работа 19.1. "Основы веб-разработки".

## Функционал.
Программа содержит файл homework.html в котором сверстана страница по продаже товаров,
содержащая в себе:
- форму обратной связи
- раздел FAQ, содержащий кнопки с выпадающими блоками
- таблицу заказов

В файле main.py создано простое web-приложение, возвращающее html страницу

## Требования к окружению

#### В программе используется менеджер зависимостей Poetry.
Используются следующие зависимости:
- python = "^3.11"
